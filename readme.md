Clone the repo
1. Install the local requirements: `npm install`

2. Install the global requirements: `$ npm install gulp bower gulp-clean gulp-jshint gulp-uglify gulp-minify-css gulp-connect --save`

3. Install the Bower components: `bower install`

4. Install bower `$bower install angular angular-animate angular-route jquery animate.css bootstrap fontawesome --save`

5. Run locally: `gulp`

6. Create a build: `gulp build`
